                    # 美股收費版

                    【 美股收費版 | 全自動股票管理表 】

                    # 連結

                    • 預覽用 美股收費版open in new window

                    # 功能

                    1. 取價改為美股網站

                    2. 整份試算表的幣別為「美金」

                    3. 儀表板 - 新增可查詢【道瓊指數】【那斯達克】【費半指數】【美股個股】的簡易資料

                      費半指數

                      此指數目前google方的圖表資料有誤，只有下面數字。沒有圖表

                    4. 儀表板新增美金兌台幣的匯率

                    5. 股票代號採手動自行更新

                    6. 平倉功能

                      平倉功能

                      參考 分頁 > 交易紀錄 > 平倉

                    # 費用

                    $ 3,000。買斷制，限用於一個Google帳號

                    # 使用方式

                    1. 購買 - 參考 付款方式
                    2. 啟用 - 參考 指南 > 開始使用

                    # 付款方式

                    有 2 種付款方式

                    1. 進入 > 商城連結open in new window > 選好商品後進入結帳頁面
                    2. 填寫資料 > Email 請填入要使用的人的 Gmail 帳號
                    3. 可選擇付款方式 ex.信用卡
                    4. 完成後續結帳步驟
                    點選展開 > 結帳流程示意圖

                    除了續訂閱之外，付款完成待懶魚確定訂單後，會發送通知信件到指定的 Gmail 信箱
